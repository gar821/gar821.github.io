Understanding Binary

What is Binary?

The binary number system is a base-2 numeral system that uses two symbols; 0, and 1.

How Does Binary Work?

In binary, each digit represents a power of 2, starting from 20 on the right. The value of the binary number is the sum of these powers of 2 where there is a 1.

Key Features of Binary:

  • Base-2 System: Uses only two digits: 0 and 1.
  • Foundation of Computing: is essential for digital electronics & computer systems
  • Addition & Subtraction is possible
